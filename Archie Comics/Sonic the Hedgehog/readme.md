# Sonic the Hedgehog (Archie Comics)

Pre-IDW era of Sonic the Hedgehog comics reading orders. There is three orders so people can decide which one they wish to take. They are taken from:

[Cobysev Read Order](https://cobysev.net/archiesonictext.html)

[Thanks Ken Penders Read Order](https://thankskenpenders.tumblr.com/readingorder)

Cobysev seemingly have most of the comics, including FCBDs. ThanksKenPenders has Megaman and Sonic X included in the Full read order, with all numbered optionals labelled ( ). No-Optionals will not have SonicX, Megaman or anything with ( ) in it from the read order.

Which is the best read order? No clue! But if you have the definitive read order, feel free to drop by the discord and demand Riju to create it.

Note: All of the Series Name in the CBL are from mylar processing, with the exception of the Sonic the Hedgehog (MiniSeries), it was renamed manually to avoid merges in Kavita. (It has the same series name and year as the main series)
